
# Coders of Bangalore — Instagram Follower Insights

A data analysis project built using pure Python and Jupyter Notebook that transforms raw, unstructured Instagram profile data into structured JSON and extracts meaningful insights from it.

This project simulates a real-world data engineering and analytics workflow by parsing noisy text-based social media data, cleaning and structuring it, and performing exploratory analysis to identify trends and patterns across Instagram accounts.

## Key Features

- Parse raw Instagram profile data from plain text format
- Convert unstructured data into structured JSON records
- Handle large numerical values with K/M suffixes (e.g., 12.5K, 1.2M)
- Identify the account with the maximum number of posts
- Find the account with the highest follower count
- Determine the account following the maximum number of users
- Analyze account categories and their distribution
- Generate insights using Python data structures and algorithms

2. Parse and extract relevant fields
