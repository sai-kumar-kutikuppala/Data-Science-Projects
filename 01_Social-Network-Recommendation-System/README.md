# Social Network Recommendation System using Pure Python

## Project Overview

This project simulates a social networking platform recommendation engine using only Python built-in libraries. The project analyzes user connections and interests to generate personalized recommendations.

## Features

* Load and parse JSON social network data
* Clean and preprocess messy datasets
* Remove duplicate and inactive records
* Implement "People You May Know" recommendations using mutual friends
* Implement "Pages You Might Like" recommendations using collaborative filtering
* Generate ranked recommendations based on similarity scores

## Technologies Used

* Python
* JSON
* Data Structures (Lists, Dictionaries, Sets)

## Project Modules

1. Data Loading
2. Data Cleaning and Preprocessing
3. People You May Know Recommendation System
4. Pages You Might Like Recommendation System
